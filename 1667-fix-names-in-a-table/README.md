<pre style="user-select: auto;">+----------------+---------+

+----------------+---------+
| user_id        | int     |

This table contains the ID and the name of the user. The name consists of only lowercase and uppercase characters.
</pre>

<p style="user-select: auto;">&nbsp;</p>

<p style="user-select: auto;">Write an SQL query to fix the names so that only the first character is uppercase and the rest are lowercase.</p>

<p style="user-select: auto;">Return the result table ordered by <code style="user-select: auto;">user_id</code>.</p>

<p style="user-select: auto;">The query result format is in the following example.</p>

<p style="user-select: auto;">&nbsp;</p>
<p style="user-select: auto;"><strong class="example" style="user-select: auto;">Example 1:</strong></p>

<pre style="user-select: auto;"><strong style="user-select: auto;">Input:</strong> 
Users table:
+---------+-------+
| user_id | name  |
+---------+-------+
| 1       | aLice |
| 2       | bOB   |
+---------+-------+
<strong style="user-select: auto;">Output:</strong> 
+---------+-------+
| user_id | name  |
+---------+-------+
| 1       | Alice |
| 2       | Bob   |
+---------+-------+
</pre>
